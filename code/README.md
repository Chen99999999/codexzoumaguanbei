# 第21届智能车竞赛 走马观碑组 项目说明

## 项目简介
本项目用于第21届全国大学生智能汽车竞赛走马观碑组。

当前项目基于一份去年旧工程进行改造，目标是复用已有的：
- 摄像头采集链路
- 图像显示/图传链路
- 双电机差速控制
- 初始化与中断框架

并在此基础上新增今年赛题所需的：
- 红色引导块检测
- 目标板 ROI 提取
- 目标板分类
- 状态机与任务策略

---

## 当前硬件情况
- 主控平台：龙芯 LS2K0300
- 转向方式：双电机差速转向
- 无舵机
- 车模前轮为支撑轮，不参与转向
- 摄像头：USB 摄像头
- 显示：屏幕已接入
- 编码器：双编码器闭环
